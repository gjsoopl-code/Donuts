# โค้ดเว็บ ร้านขายขนมปัง รวมมิตร

ล่างนี้คือโครงสร้างทั้งหมด

/bakery
├── index.php        (หน้าหลัก: แสดงสินค้า, ตะกร้าสินค้า, ระบบสั่งซื้อ)
├── admin.php        (หน้าหลังบ้าน: เพิ่ม/ลบสินค้า, ดู/ลบ Log การซื้อ)
├── products.json    (ฐานข้อมูลสินค้า: เก็บชื่อ, ราคา, สต็อก, ลิงก์รูปภาพ)
├── orders.json      (ฐานข้อมูลคำสั่งซื้อ: เก็บที่อยู่, รายการที่ซื้อ, ยอดรวม)
└── /images          (ทางเลือก: สำหรับเก็บไฟล์รูปภาพถ้าไม่ใช้ลิงก์ URL)

🔄 แผนผังการทำงาน (Workflow)
​1. ฝั่งลูกค้า (index.php)
​อ่านข้อมูล: ดึงข้อมูลจาก products.json มาแสดงเป็น Card สินค้าสวยๆ
​เลือกสินค้า: ใช้ JavaScript เก็บข้อมูลสินค้าที่เลือกไว้ในตะกร้า (SessionStorage)
​สั่งซื้อ: เมื่อกด "ยืนยันสั่งซื้อ" ระบบจะ:
​ส่งข้อมูลที่อยู่และรายการสินค้าไปที่ PHP
​PHP เข้าไปลดจำนวน stock ใน products.json
​PHP เพิ่มข้อมูลการสั่งซื้อใหม่ลงใน orders.json
​2. ฝั่งเจ้าของร้าน (admin.php)
​จัดการสต็อก:
​เพิ่ม: กรอกข้อมูลผ่านฟอร์ม -> PHP เขียนลง products.json
​ลบ: กดปุ่มลบ -> PHP ค้นหา ID สินค้าแล้วตัดออกจาก products.json
​จัดการ Log: * ดึงข้อมูลจาก orders.json มาแสดงตามลำดับล่าสุด
​สามารถลบทีละรายการ หรือล้างประวัติทั้งหมดได้
